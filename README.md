# onlyEffort

V1 : 알고리즘 스터디원들의 학습패턴을 상세히 분석해주자.

1. Account 기능
   - 약관동의
   - 회원가입
   - 회원탈퇴
   - 비밀번호 변경
   - 로그인
   - 로그아웃
2. Admin 페이지
   - 회원가입 수락/거절
   - 글관리 / 회원관리
3. 건의사항 페이지
   - CRUD
   - Private 지원
   - Status 관리
4. 학습패턴 분석페이지
   - 일간 분석
   - 주간 분석
   - 우수 스터디원 선정
